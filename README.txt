========================================
CXSTOMER 5th Anniversary - Website
========================================

This is a simple HTML/CSS/JavaScript website that you can easily edit.

FILES:
------
- index.html    - Main website file (contains HTML, CSS, and JavaScript)
- venue-luzon.jpg       - Venue image for Luzon
- venue-visayas.jpg     - Venue image for Visayas  
- venue-mindanao.jpg    - Venue image for Mindanao
- tropical-leaves.png   - Background decoration

HOW TO EDIT:
------------
1. Open index.html in any text editor (Notepad, VS Code, Sublime, etc.)
2. Find the text you want to change
3. Edit it and save the file
4. Open index.html in a browser to see changes

WHAT YOU CAN EASILY EDIT:
-------------------------

1. EVENT DATES:
   Search for "April 18, 2026", "May 9, 2026", or "May 23, 2026"

2. VENUE NAMES:
   Search for "Casa Alessandra", "Shangri-La Mactan", or "Eden Nature Park"

3. VENUE ADDRESSES:
   Search for "Mexico Pampanga", "Lapu-Lapu City, Cebu", or "Toril, Davao City"

4. EVENT TITLES:
   Search for "LUZON TAKEOVER", "VISAYAS GATHERING", or "MINDANAO MEETUP"

5. PROGRAM SCHEDULE:
   Look for the "Program flow" sections and edit the times

6. FAQS:
   Find the FAQ sections and edit questions/answers

7. CONTACT INFO:
   Search for "Blad, KM, HEs, or Pearl" or "#cxstomer-events"

8. IMAGES:
   Replace venue-luzon.jpg, venue-visayas.jpg, or venue-mindanao.jpg
   with your own images (keep the same filenames)

COLORS (in the CSS section):
----------------------------
--jungle: #0B3A2E     (dark green background)
--ivory: #F3EFE6      (light background)
--gold: #D4A02C       (accent color)
--offwhite: #F7F7F5   (text color)

HOW TO UPLOAD TO WEB:
---------------------
1. Upload all files to your web hosting
2. Make sure all files are in the same folder
3. The main page is index.html
